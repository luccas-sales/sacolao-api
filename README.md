# 🥬 API do Sacolão

API desenvolvida para centralizar e estruturar operações internas da rede de mercados Sacolão.

Construída com foco em organização, escalabilidade e padronização de dados, servindo como base para diferentes funcionalidades e integrações.

---

## 🚀 Tecnologias utilizadas

- NestJS
- TypeScript
- PostgreSQL
- Prisma ORM
- Supabase

---

## 📦 Funcionalidades **ATUAIS**

- CRUD de fornecedores
- CRUD de descrições de fornecedores
- Integração com banco de dados PostgreSQL
- Estrutura modular escalável
- Arquitetura organizada com NestJS

---

## 📈 Objetivo do projeto

Facilitar automações internas do dia a dia.

Atualmente, a API é utilizada como base para um sistema de gerenciamento de fornecedores e será expandida para suportar novas automações e fluxos dentro do negócio.

---

## 🔐 Observações

- Este projeto **não contém dados sensíveis**
- Não possui autenticação (JWT, etc.) no momento
- Projeto público com finalidade de estudo, portfólio e automações internas

---

## 📁 Estrutura

O projeto segue a arquitetura padrão do NestJS, com separação por módulos, facilitando manutenção e escalabilidade.

---

## 💡 Futuro

A API está em constante evolução e será expandida com novas funcionalidades e integrações conforme necessário.
